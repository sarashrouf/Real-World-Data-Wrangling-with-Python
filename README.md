# Data Wrangling Project

This project focuses on gathering, cleaning, and analyzing movie datasets from multiple sources.

## Key Steps:
- Data Gathering (API + manual)
- Data Cleaning
- Data Merging
- Exploratory Data Analysis

## Key Insights:
- Netflix movies tend to have slightly higher ratings
- Movie duration follows a common pattern (80–120 mins)

## Tools Used:
- Python
- Pandas
- Matplotlib
